# Local Knowledge Demo Notes

This file is synthetic demo material for public documentation and screenshots.

## Product purpose

Local Knowledge Demo shows how a local document source can become searchable evidence.
The demo uses a local source folder, a scan step, extraction into citable segments, search results, and a grounded answer response.

## Supported demo behavior

The demo should show that search results include document references and citation labels.
The demo should show that grounded answers are based on retrieved evidence instead of hidden free-form guessing.
The demo should show a clean no-evidence result when a question has no matching local source.

## Local-first boundary

The demo path is designed for local use.
The default demo does not require uploading documents to a remote service.
Optional local LLM and browser layers can be enabled separately when needed.
