# Demo Support FAQ

This file contains artificial support information for public demo use.

## How do I start the stack?

Use the beginner start path for your operating system, or use the Python entrypoint when you want the same command style across platforms.
After startup, run the health check before testing search or grounded answers.

## What should I do if search returns no results?

First confirm that the source folder was scanned.
Then confirm that the expected files were extracted into segments.
Finally, try a question that uses words known to appear in the demo files, such as vacation, citation, or screenshots.

## What should a no-result answer do?

A safe no-result answer should say that no reliable evidence was found.
It should not invent facts that are missing from the local source material.

## What is safe to screenshot?

It is safe to screenshot the synthetic demo files, search results from those files, and grounded answers based on those files.
It is not safe to screenshot private documents, secrets, customer data, private logs, or local machine paths that reveal personal information.
