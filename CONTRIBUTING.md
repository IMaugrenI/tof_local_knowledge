# Contributing

Thank you for your interest in this repository.

## Scope first

Before proposing changes, read these files first:

- `README.md`
- `START_HERE.md`
- `docs/00_product_scope.md`
- `docs/01_architecture.md`
- `docs/11_runtime_stack.md`

## What to preserve

Changes should preserve these repository characteristics:

- local-first / on-prem reading
- grounded answers from local evidence
- separation between raw input, extraction, index, and answer layers
- clear boundaries between source material and generated output
- explicit documentation instead of hidden behavior

## Pull request expectations

Contributions should be small, traceable, and scoped.

Please prefer:

- focused pull requests
- direct wording in docs
- minimal hidden magic
- boundary-preserving changes
- updates to documentation when behavior changes

## Do not assume

Do not assume that this repository represents the whole private ToF / V’eth project line.
Do not assume that terms or boundaries from other repositories automatically apply here.

## Legal note

Check `LICENSE` and any additional repo notes before reusing code or documentation.
