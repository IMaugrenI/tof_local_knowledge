# MVP-Plan

## MVP-Ziel
Schnell real nutzbares lokales Dokumentfinden mit evidenzbasierter QA.

## MVP-Inhalt
- 1 bis wenige lokale Quellen
- 1 Admin + kleine Nutzergruppe
- Scan read-only
- Kernformate extrahieren
- Volltext + Filter
- Antworten mit Fundstellen
- Grundrollen
- Grundaudit

## Nicht im MVP
- OCR-Vollausbau
- Mandantenbetrieb
- Auto-Cluster
- Auto-Glossar
- verteilte Sync-Topologie
